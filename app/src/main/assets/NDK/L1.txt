
------ 知识点 ------

必须有一个 main 函数；

打印出一个基本类型变量语句：
short %d
long %ld
float %f
double %c
char %c

获取一个输入的数据并赋值给某个变量
scanf（"%d",&i）

打印一个变量内存地址
printf("dizhi %#x\n",&i);

定义一个数组必须在【编译】时就确定数组的大小： int xArray[10] 
以下情况不行：
1、设置一个 成员变量 给成员变量赋值 数组的大小是这个成员变量的值 编译不通过
2、获取scanf输入的值 设置给成员变量
给成员变量设置一个 const 关键字 这个情况可以 相当于 java 的 static。c在编译期就在静态内存区域分配了内存空间

为什么要确定数组的大小？
C语言定义数组时候必须把数组定义在一块连续的空间内。所有在编译期间就要明确大小并分配好内存空间。

栈的大小是系统设定好的 默认 10M

在堆里面动态申请的空间系统是不会释放的 需要手动释放空。delete（xx）



sizeof(int) 计算基本数据类型占字节的大小 跟变量没关系 根据不同系统所占字节大小不一样。sizeof是个关键字 不是函数

----------------------------
指针
操作符 & 取地址操作符。只能运用于变量跟数组元素。
操作符 * 间接询址。


指针只是一个存储 内存地址 的变量， 内存地址只占 4 个字节（所占字节跟系统有关不一定是 4）。
